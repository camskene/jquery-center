jQuery Center Plugin

Centers any element to the window or it's parent element.

By default element is centered horizontally and vertically.

Options

There are two options.

                vertical: true,
                horizontal: true

1. Set vertical to false to center element horizontally only.

2. Set horizontal to false to center element verically only.

HTML

JS

                $("foo").center()

or

                $("foo").center({
                    vertical: true,
                    horizontal: true
                })
